# Jornadas

## Criar conta

Usuário acessa a plataforma
↓
Seleciona a opção de criar conta
↓
Informa seus dados pessoais
↓
Sistema valida os campos obrigatórios
↓
Sistema verifica se o CPF já está cadastrado
↓
Sistema verifica se o e-mail já está em uso
↓
Sistema cria o cadastro do usuário
↓
Sistema cria a conta bancária vinculada
↓
Sistema confirma a criação da conta

---

## Fazer login

Usuário informa e-mail e senha
↓
Sistema localiza o usuário
↓
Sistema valida as credenciais
↓
Sistema autentica o usuário
↓
Sistema gera um token de acesso
↓
Sistema libera o acesso à conta

---

## Fazer Pix

Usuário acessa a área Pix
↓
Informa a chave Pix do destinatário
↓
Informa o valor da transferência
↓
Sistema verifica se a chave Pix existe
↓
Sistema verifica se há saldo suficiente
↓
Usuário confirma a operação
↓
Sistema realiza a transferência
↓
Sistema registra a movimentação financeira
↓
Sistema confirma o pagamento

## Solicitar cartão

Usuário acessa a área de cartões
↓
Solicita um novo cartão
↓
Sistema verifica se a conta está ativa

↓
