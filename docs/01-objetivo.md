```md
# Objetivo

## Visão

O AlexandriaBank é um banco digital fictício criado para simular o funcionamento de uma plataforma bancária moderna.

## Problema

Simular um sistema bancário que permita realizar operações financeiras de forma simples, segura e totalmente digital, reduzindo burocracias presentes em processos tradicionais.

## Objetivo de estudo

Construir um projeto de longo prazo para desenvolver minhas habilidades em engenharia de software, aprendendo a pensar no sistema antes da implementação.

Ao longo da evolução do projeto, o foco será praticar:

- Levantamento de requisitos
- Jornadas de usuário
- Modelagem de domínio
- Arquitetura de software
- Tomada de decisões arquiteturais
- Evolução da arquitetura conforme o crescimento do sistema
- Escalabilidade
- Observabilidade
```
